# Project LawuPOS
### Ini adalah sistem POS / Kasir open-source yang dibangun menggunakan SvelteKit, ElysiaJS, dan PostgreSQL
---
### Panduan Instalasi:
## 1. Clone
Clone repositori ini dengan
```
git clone https://github.com/rzlmiooo/sistem-kasir.git
```
## 2. Instalasi Frontend
Masuk ke folder frontend
```
cd frontend
```
kemudian
```
npm install 
``` 
atau
```
pnpm install 
``` 
atau 
```
bun install
```
lalu buat dan konfigurasikan file ```.env```
```
VITE_API_DIR_PATH=http://localhost:5173/api/interceptor
API_URL=http://localhost:3000
```

## 3. Instalasi Backend
Masuk ke folder backend
```
cd backend
```
kemudian
```
npm install 
``` 
atau
```
pnpm install 
``` 
atau 
```
bun install
```
